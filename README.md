# Servey Project

## Project Stacks

### Backend
  - Node.js
  - Koa + TypeORM
  - PostgreSQL

### Frontend
  - React
  - Mobx
  - React Router
  - Axios
  - Styled Components
  - Parcel

### Shared
  - Test Tool: Jest
  - Language: Typescript
  - I/O Model : shared directory
  - Environment: Docker


## 개발용 서버 실행
```
$ docker-compose up -d postgres-server
$ docker-compose run shell
    # yarn install
    # exit
$ docker-compose up -d server
```
